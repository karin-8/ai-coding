# AI and Coding Workshop 🤖💻

Welcome to the **AI and Coding Workshop** repository. This project contains essential resources and hands-on materials designed to help you leverage modern AI tools to build, experiment, and "vibe code" your way through development.

## 📁 Repository Contents

* **`AI and Coding Workshop.pdf`**: The primary presentation and guide covering the theoretical and practical intersections of AI and software development.
* **`Worksheet Vibe Coding Hands-On.pdf`**: A practical, step-by-step worksheet to guide you through a hands-on session using AI-assisted coding techniques.

## 🚀 What is "Vibe Coding"?

"Vibe Coding" refers to a high-level, intuitive approach to programming where you use natural language and AI collaborators to handle the syntax and architecture, allowing you to focus on the overall logic, creative vision, and "vibe" of the project.

## 🛠️ Getting Started

1.  **Review the Workshop PDF**: Start with the main slide deck to understand the core concepts.
2.  **Complete the Worksheet**: Open the "Vibe Coding Hands-On" file and follow the instructions to build your first AI-assisted project.
3.  **Experiment**: Use the techniques learned here to automate repetitive tasks or prototype new ideas quickly.
